# Compensation Planning Overview

<Var name="plannerName" /> provides a comprehensive compensation planning tool for managers and HR teams.

## Key Features

- **Budget allocation** across departments and teams
- **Merit increase** recommendations based on performance
- **Bonus calculations** using configurable formulas
- **Equity awards** management and vesting schedules

<MessageBox type="tip">

Use the simulation mode to preview the impact of your compensation decisions before submitting for approval.

</MessageBox>

## Getting Started

Access <Var name="plannerName" /> from the main navigation menu. Your available planning cycles will be displayed on the dashboard.
